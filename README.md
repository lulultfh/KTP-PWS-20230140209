# Project Structure
```
├── db/
│   ├── db.js                          # Konfigurasi dan inisialisasi koneksi ke MySQL
├── public/
│   ├── index.html
│   ├── style.css
├── routes/                            # Routing API untuk CRUD
│   ├── ktp-db.js
├── uploads/                           # Untuk menyimpan gambar yang diunggah
├── .env
├── index.js                           # Entry point server Express
└── README.md                          # Dokumentasi utama proyek
```
